 Comparative Framework for Diabetes Prediction Using Balanced Machine Learning and Deep Learning Models 
 
## ?? Overview 
 
This project evaluates **7 models** for diabetes prediction using the BRFSS 2015 dataset: 
 
- **Machine Learning**: Logistic Regression, Na‹ve Bayes, Random Forest, XGBoost 
- **Deep Learning**: ANN, CNN, RNN 
 
## ?? Key Results 
 
 
## ?? Dataset 
 
- **Source**: CDC BRFSS 2015 
- **Records**: 253,680 
- **Classes**: No Diabetes, Prediabetes, Diabetes 
- **Features**: 21 health indicators
